링크설명   target 속성   경로값(url)의 범위

링크설명▲

a태그 : 텍스트나 이미지를 통하여 다른 곳으로 정보를 연결해 주는 역할을 담당하는 요소입니다

a는 “anchor”의 약자이다.

사용자가 커서를 웹 페이지의 링크 위에 올리면 커서의 모양이 화살표에서 손 모양으로 변화한다.

링크 주소는 href속성으로 정의된다.

target 속성▲

"_blank" : 새로운 창으로 열립니다.

"_self" : 현재 윈도우에 새로운 페이지를 적재합니다. 기본값

예: 네이버

경로값(url)의 범위▲

절대경로 : 다른 웹 사이트의 페이지, http://www.naver.com

상대경로 : 웹 사이트 안에서의 다른 페이지, ./4 텍스트.html

내부파일 : 현재페이지 안의 다른 위치,id값으로 연결한다

상대경로의 예 : 텍스트 페이지로 이동
