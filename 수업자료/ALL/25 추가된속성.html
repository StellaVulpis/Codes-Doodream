*표시는 필수입력 항목입니다.

필드 유효성 검사

  • required : 입력 양식을 제출하기 전에 반드시 채워져 있어야 함을 나타낸다.
  • placeholder : 입력 힌트를 희미하게 보여준다.
  • autocomplete속성은 자동완성 기능에 대한 사용 여부를 판단해 줍니다.
    자동완성 기능을 사용하고 싶으면 속성값을 'on'으로, 아니면 'off'를 주면 됩니다
  • 페이지가 로드되었을 때 입력 필드에 포커스(커서)가 자동으로 이동하게 해 주는 속성입니다.
  • disabled : 비활성화 상태 만들기
  • readonly : 읽기 전용 필드, input요소의 값이 변경되지 않은 채 폼 전송이 되어야 하는 곳에 사용합니다
  • pattern : 허용하는 입력의 형태를 정규식으로 저장한다.
